#  Terminal Chat App with Redis Stack

Легкий и быстрый консольный многопользовательский чат, написанный на Python. Проект демонстрирует мощь и возможности **Redis**, используя его не просто как кэш, а как полноценную базу данных и брокер сообщений с применением продвинутых структур данных (Фильтры Блума, Битовые карты, Хеши и т.д.).

## ✨ Возможности и архитектура (Использование Redis)

В проекте применяются различные структуры данных Redis для решения конкретных архитектурных задач:

*   **Pub/Sub (Издатель/Подписчик):** Мгновенная доставка сообщений всем подключенным клиентам в реальном времени.
*   **Bloom Filters (Фильтры Блума):** Сверхбыстрая проверка уникальности никнеймов с минимальным потреблением памяти (модуль `RedisBloom`).
*   **Hashes (Хеши):** Хранение профиля пользователя (внутренний ID и зашифрованный SHA-256 пароль).
*   **Bitmaps (Битовые карты):** Память-эффективный подсчет уникальных активных пользователей за день (Daily Active Users - DAU). Состояние миллиона пользователей занимает всего ~120 КБ!
*   **Lists (Списки):** Хранение истории чата. Реализован паттерн "Круговой буфер" (Ring Buffer) через `RPUSH` и `LTRIM` — сохраняются только последние 200 сообщений.
*   **Sets (Множества):** Отслеживание пользователей, которые находятся онлайн *прямо сейчас*, что предотвращает одновременный вход под одним ником с разных терминалов.

## 🛠 Требования

*   **Python 3.7+**
*   Установленная библиотека `redis` для Python.
*   **Redis Stack Server** (Обычный Redis не подойдет, так как нам нужны модули для работы с Фильтрами Блума).

## 🚀 Установка и запуск

### 1. Запуск Redis Stack
Вам нужно запустить сервер Redis Stack на стандартном порту (6379).

**Вариант 1: Через Docker**
```bash
docker run -d --name redis-chat -p 6379:6379 redis/redis-stack-server:latest
```

**Вариант 2: Установка на Ubuntu / WSL**
Если у вас запущен обычный Redis, сначала остановите его: `sudo service redis-server stop`. Затем установите Redis Stack:
```bash
curl -fsSL https://packages.redis.io/gpg | sudo gpg --dearmor -o /usr/share/keyrings/redis-archive-keyring.gpg
echo "deb [signed-by=/usr/share/keyrings/redis-archive-keyring.gpg] https://packages.redis.io/deb jammy main" | sudo tee /etc/apt/sources.list.d/redis.list
sudo apt-get update
sudo apt-get install redis-stack-server
redis-stack-server
```

### 2. Установка зависимостей Python
Клонируйте репозиторий и установите пакет `redis`:
```bash
pip install redis
```

### 3. Запуск чата
Откройте терминал и запустите скрипт:
```bash
python chat.py
```
*💡 Совет: Откройте 2-3 разных окна терминала и запустите скрипт в каждом из них под разными никнеймами, чтобы протестировать общение в реальном времени.*

## 🎮 Как пользоваться

1. При первом вводе никнейма система распознает вас как нового пользователя и попросит придумать пароль (символы при вводе скрыты).
2. При повторном входе система попросит ввести пароль для авторизации.
3. Пишите сообщения и нажимайте `Enter`. Они мгновенно появятся у всех остальных пользователей.
4. Для выхода из чата напишите `/quit` или нажмите `Ctrl+C`.

## 🔧 Полезные команды для разработчика

Если вы хотите полностью сбросить базу данных (удалить всех пользователей и историю сообщений), выполните в терминале команду:
```bash
redis-cli flushall
```

Чтобы проверить, загрузились ли нужные модули (например, Фильтр Блума) в вашем Redis:
```bash
redis-cli info modules
```
